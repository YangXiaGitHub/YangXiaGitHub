三. 多个ssh key

  1. 背景：如果～/.ssh文件夹下面已经有了其他的ssh key，这时候你在命令行输入

ssh-keygen -t rsa

会导致原来的ssh key被覆盖，那么以前的key就会失效。

  1. 解决方案：让多个key共存

  2. 解决办法：

    第一步

    ssh-keygen -t rsa -f ~/.ssh/github_id-rsa

    上面的命令会在～/.ssh下面生成一对名字为github_id-rsa的公钥和密钥，不与原来的发生冲突。
